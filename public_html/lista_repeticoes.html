Exercicio

  1. Crie um campo em que o usuário possa informar um número;
  2. Adicione um evento em que após pressionar uma tecla (onkeyup) chama uma função que calcula o fatorial do número informado;
  3. O programa deve exibir o processo de calculo em uma tag de tabela ( < table > ) como a tabela abaixo demonstra:
    Exemplo de resultado se o número 5 for informado pelo usuário:
    IndiceMultiplicadorResultado
    1 1 1
    2 1*2 2
    3 1*2*3 6
    4 1*2*3*4 24
    5 1*2*3*4*5 120
  4. Se o usuário informar um número negativo, converte-lo para positivo antes de realizar o calculo
  5. Crie o campo para informar o número e a tabela que deverá ser preenchida dentro do quadro amarelo do exercício.

Exercicio

O botão "Verificar estoque" dispara uma função javascript verifica_estoque() que obtem um array de tags com o valor de estoque de cada produto em uma almoxarifado.

  1. Modifique a função para apresentar a quantidade em estoque em vermelho se estiver abaixo de 0, mostrar em cinza se estiver igual a 0 e verde se estiver com valor acima de 0;
  2. Adicione mais uma coluna à tabela com um botão "Adicionar ao estoque" em cada linha de produto;
  3. O botão "Adicionar ao estoque" deve adicionar à quantidade em estoque da linha em questão o valor de 10 unidades.
    Para faze-lo, a função que o evento de onclick chama deve enviar como parametro a posição do item na tabela (exemplo: adicionar_ao_estoque(3) se o botão estiver na quarta linha da tabela);
