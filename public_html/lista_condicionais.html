Exercicio

Escrever um algoritmo que leia três notas obtidas por um aluno durante o curso. Calcular a sua média (aritmética). Apresentar no .quadro sua situação dentre as possíveis:

  • Maior ou igual a 7: Aprovado
  • De 5 até 7: De recuperação
  • Menor do que 5: Reprovado

Aqui será apresentado o resultado

Exercicio

Escrever um algoritmo para verificar se o aluno será reprovado por falta levando em consideração os seguintes pontos:

  1. Criar um campo em que o usuário informa a quantidades de aulas (horas) que todo o curso é composto. (obs: adicionar um valor definido por padrão, ou seja, preenchido previamente)
  2. Criar um campo em que o usuário informa a quantidade de faltas que possuí (aulas, não dias)
  3. Criar um botão que deve realizar as seguintes operações:
    • Sabendo que a cada dia de curso, são registradas 4 presenças (aulas), calcular a quantidade de dias que o aluno faltou;
    • Para que o aluno seja aprovado é necessário que esteja presente em 75% das aulas. Portanto, deve ser calculado qual é a porcentagem de presença do aluno;
  4. Apresentar o resultado dos calculos do item anterior na página dentro do elemento (tag) #resultados . Levando em consideração os seguintes requisitos
    • Criar um quadro para apresentar a quantidade de dias em que o aluno faltou;
    • Criar um quadro para apresentar a percentagem de presença;
    • Se o aluno estiver reprovado por falta, apresentar na página o seguinte: "O aluno está reprovado por falta". Se estiver aprovado, não apresentar mensagem alguma;
    • Se o aluno estiver reprovado, a porcentagem de falta deve estar em cor vermelha;
    • Se o aluno estiver aprovado, a porcentagem de falta deve estar em cor verde;
