Exemplo 1

Desenvolver um site segundo características descritas abaixo.
No dia Entrega deverá ser feita

  1. Site deverá ser composto de pelo menos 4 TIPOS de página. Ex:
    • Página de listagem de itens, como uma loja, blog, porfolio pessoal, catáogo, etc...
    • Página de contato/cadastro, contendo um formulário que deve ser validado via javascript (detalhamento posterior)
    • "Landing page" (Ver referências no google), página que apresenta a loja, normalmente diferente do resto do site, onde ao final há uma "chamada" (detalhamento posterior).
    • Página de perfil, exibindo informações pessoais do responsável pela página, ou dos usuários cadastrados no site.
    • Página de detalhamento. Exibindo imagens, especificações técnicas, demonstrações, vídeos relativos à um produto ou serviço.
    • Página de login, podendo ser também uma página de cadastro, separado por colunas ou linhas.
    • Página de compra, contendo dados dos itens que se deseja comprar, formas de pagamento, etc...
  2. Todas as páginas deverão ter vísivel um menu, com links para as principais páginas.
    ex: contato, home, catalogo, produtos, cadastre-se, empresa, serviços, etc...
  3. Todas as páginas deverão conter um rodapé com pelo menos duas colunas.
    • Uma que DEVE apresentar o logotipo e links reelevantes. Tais links, podem ser os mesmos do menu, links para mídias sociais, telefone, e-mail para contato, endereço, ou qualquer outra informação sobre o negócio.
    • Uma coluna ou mais para apresentar cada integrande do desenvolvimento do trabalho, exibindo pelo menos nome e e-mail de cada um.
  4. Todas as páginas deverão utilizar um mesmo CSS para estilização. Entretanto, é possível que páginas tenham seus proprios arquivos A MAIS do que o arquivo padrão.
  5. Deve ser utilizado em alguma página javascript para validação ou automatização de alguma coisa. Ex: validar campos de formulário, mudar cores de produtos, mudar imagens, etc...
  6. É permitido e incentivado o uso do jQuery sempre que possível. (1/10 da nota como bonus para utilização mais que o especificado acima)
  7. É Permitido o uso de frameworks CSS como o twiter bootstrap, Materialize, entre outros
